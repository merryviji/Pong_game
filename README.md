A simple 2D Pong game built using Python’s Turtle graphics module. This project recreates the classic arcade-style game with two paddles and a bouncing ball.

🎮 Features
Two-player paddle controls
Ball movement with collision detection
Score tracking system
Increasing speed for added difficulty

🛠️ Built With
Python
Turtle (built-in graphics library)

🚀 How to Run
Make sure Python is installed on your system

Clone this repository:

git clone https://github.com/merryviji/pong-game.git

Navigate to the project folder:

cd pong-game

Run the game:
python main.py


🎯 Controls
Left Paddle: W (up), S (down)
Right Paddle: Up Arrow (↑), Down Arrow (↓)
